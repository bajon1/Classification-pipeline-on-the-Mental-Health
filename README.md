# Mental Health in Tech - Classification Pipeline

End-to-end deep learning pipeline for predicting mental health treatment-seeking behaviour
among tech workers, built with PyTorch, Optuna, and TensorBoard.
 
---
 
## Description
 
Mental health remains underreported and poorly understood in the tech industry.
This project builds a complete, production-style machine learning pipeline on the
[OSMI Mental Health in Tech Survey](https://www.kaggle.com/datasets/osmi/mental-health-in-tech-survey)
dataset, a real-world, messy, imbalanced dataset that reflects the challenges of
working with human survey data.
 
The goal is not just to train a model, but to demonstrate the full engineering
process: from raw data exploration and principled feature cleaning, through
architecture search with cross-validated hyperparameter optimisation, to ensemble
inference with uncertainty estimation and statistical comparison of model variants.
